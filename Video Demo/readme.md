## 🚀 Key System Features Mapped in the Demo
1. **Relational Data Schema:** Custom objects (`Job_Opening__c` and `Candidate_Application__c`) connected via dual Master-Detail Relationships to ensure high data integrity.
2. **Data Cleanliness & Validation Rules:** Strict database validation formulas preventing invalid data inputs, such as negative vacancy targets and positive offer constraints ($>0$).
3. **Salesforce Flow Automations:** Record-Triggered Flows for high-priority scoring notifications (AI Score $\ge 85$), Screen Flows for custom entry wizards, and daily Scheduled-Triggered batch flows.
4. **Agentforce Generative AI Integration:** Natively embedded layout prompt blocks powered by the active **GPT 5 Mini model framework**, secured safely under the Einstein Trust Layer guardrails.
5. **Corporate Budgetary Governance:** Defending organizational parameters via multi-action Classic Approval Processes that automatically trigger and lock record data rows when compensation plans exceed 1,000,000 currency units.
